## 📋 用户使用流程

当其他人克隆你的项目后，他们需要：

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/chat-with-idol.git
   cd chat-with-idol
   ```

2. **配置环境变量**
   - 复制 `backend/env.example` 为 `backend/.env`
   - 填入自己的 API Key

3. **安装依赖并启动**
   - 参考 `SETUP.md` 或 `README.md`
