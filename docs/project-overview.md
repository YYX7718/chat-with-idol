# 聊天占卜偶像项目概述

## 项目背景

随着人工智能技术的发展，虚拟偶像与AI聊天系统越来越受到用户欢迎。本项目结合了聊天互动与占卜服务，为用户提供全新的虚拟偶像体验。

## 核心功能

### 1. 偶像选择
用户可以从多个虚拟偶像中选择自己喜欢的角色进行互动。每个偶像都有独特的性格设定和对话风格。

### 2. 聊天互动
用户可以与所选偶像进行自然语言对话，偶像会根据系统提示词和对话历史进行回应，提供情绪支持和娱乐。

### 3. 占卜服务
系统提供多种占卜类型（如爱情、事业、运势等），用户可以请求占卜，偶像会以个性化的方式提供占卜结果。

### 4. 会话管理
系统会保存用户与偶像的聊天历史和占卜记录，方便用户回顾和继续对话。

## 系统架构

### 后端架构
- **Flask 框架**：提供 API 服务
- **LLM 客户端**：调用 DeepSeek 或 OpenAI API
- **服务层**：处理占卜和聊天业务逻辑
- **数据模型**：管理会话数据

### 前端架构
- **Vue 3**：构建用户界面
- **组件化设计**：实现页面和功能组件
- **API 封装**：与后端进行通信

## 开发流程

1. **需求分析**：确定功能需求和技术方案
2. **设计阶段**：设计系统架构、数据库模型和界面原型
3. **开发阶段**：实现后端 API 和前端界面
4. **测试阶段**：进行单元测试和集成测试
5. **部署阶段**：部署到生产环境
6. **维护阶段**：监控系统运行并进行优化

## 项目目标

- 提供流畅自然的聊天体验
