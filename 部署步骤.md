# 🚀 快速部署步骤

## 最简单的方式：Vercel + Railway

### ⚡ 5 分钟快速部署

#### 第一步：部署后端（Railway）

1. 访问 https://railway.app，用 GitHub 登录
2. 点击 "New Project" → "Deploy from GitHub repo"
3. 选择你的仓库
4. 在项目设置中添加环境变量：
   ```
   DEEPSEEK_API_KEY=你的API密钥        # 必需：DeepSeek API密钥，用于调用AI服务
   DEFAULT_MODEL=deepseek-chat         # 可选：默认使用的AI模型（默认值：deepseek-chat）
   FLASK_DEBUG=False                   # 必需：生产环境设为False，关闭调试模式
   FLASK_PORT=5000                     # 可选：后端服务端口（Railway会自动分配，通常不需要设置）
   ```
   
   **环境变量说明：**
   - `DEEPSEEK_API_KEY`: 你的 DeepSeek API 密钥（必需），从 https://platform.deepseek.com/ 获取
   - `DEFAULT_MODEL`: 使用的 AI 模型名称，默认是 `deepseek-chat`
   - `FLASK_DEBUG`: 生产环境必须设为 `False`，开启调试模式会有安全风险
   - `FLASK_PORT`: 服务端口，Railway 会自动分配，一般不需要手动设置
5. 等待部署完成，复制后端 URL（例如：`https://xxx.up.railway.app`）

#### 第二步：部署前端（Vercel）

1. 访问 https://vercel.com，用 GitHub 登录
2. 点击 "Add New Project"
3. 选择你的仓库
4. 配置：
   - **Root Directory**: `frontend`
   - **Framework Preset**: `Vite`
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
5. 添加环境变量：
   ```
   VITE_API_BASE_URL=https://你的后端URL/api
   ```
   （将 `你的后端URL` 替换为第一步获得的后端地址）
6. 点击 "Deploy"
7. 等待部署完成，获得前端 URL（例如：`https://xxx.vercel.app`）

#### 第三步：更新 CORS

回到 Railway，更新环境变量：
```
CORS_ORIGINS=https://你的前端URL
```
（将 `你的前端URL` 替换为第二步获得的前端地址）

重新部署后端。

在 `README.md` 中更新在线演示链接：
```markdown

[👉 点击体验在线版本](https://你的前端URL)
```
